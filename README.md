# Agentiom

Deploy stateful AI agents in seconds. Persistent storage. Zero cold starts.

## Quick Start
```bash
npx agentiom init my-agent
cd my-agent
npx agentiom deploy
```

## Features

- **Persistent Storage** - Real filesystem that survives restarts
- **Wake on Trigger** - Sleep when idle, wake on email/webhook/cron
- **One Command Deploy** - From terminal to cloud in seconds
- **Isolated Runtime** - Each agent runs in its own container
